# Karo Pitch

## What is this?
A landing page for Karo Pitch - a platform where Indian founders can pitch to investors.

## How to run?
1. npm install
2. npm start

## Built with
- React.js
- CSS3

## Features
- Hero section with buttons
- 4-step process
- Founder categories
- Investors section
- Startup cards
- Mobile friendly

## Design
- Orange (#F97316) and Green (#2D6A4F) theme
- Clean modern look
